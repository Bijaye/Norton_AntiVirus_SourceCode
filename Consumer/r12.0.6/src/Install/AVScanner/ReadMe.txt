========================================================================
    DYNAMIC LINK LIBRARY : AVScanner Project Overview
========================================================================

The AVScanner project is a syminterface dll that implements an IPreFlightScanner
and IPreFlightScanner derived object.  The IPreFlightScanner derived object is 
implemented in AVScanObject.cpp and implements a PreFlight Antivirus scanner.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named AVScanner.pch and a precompiled types file named StdAfx.obj.
