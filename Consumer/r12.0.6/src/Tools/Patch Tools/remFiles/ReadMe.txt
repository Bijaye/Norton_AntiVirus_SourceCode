========================================================================
       WIN32 APPLICATION : remFiles - Tool created by Wesley Jordan for
			   the Espana LU patch to update all versions of
			   NAV 2002 to r8.07.
========================================================================

This tool will add alert.dat to the RemoveFile Table of the local NAV
MSI package.  It also updates navap.cat and navap.inf files on systems that
already have these files (only XP systems).
