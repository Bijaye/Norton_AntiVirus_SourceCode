Preconfigured packages

Symantec provides the following preconfigured installation packages for your convenience. When you click a link below, you'll be given the option to save the file or run it from its current location.

Package name Description
Symantec AntiVirus Lightly Managed Client Installs the Symantec AntiVirus Client software to the target computer. The lightly managed client only includes the following features:
  • The AMS client for alert forwarding
  • All decomposers but Ghost images
  • All help files
  • Email protection plug-ins for Lotus Notes and Microsoft Exchange/Outlook
  • Scan and Deliver
  • File system real-time protection
  • Live Update

You can manage these clients by rolling out a new Grc.dat using Symantec Packager or your own third-party software.

Symantec AntiVirus Thin Client Installs the Symantec AntiVirus Client software to the target computer. The thin client only includes the following features:
  • ZIP decomposer only. All other decomposers are excluded.
  • All help files
  • File system real-time protection
  • Live Update
Symantec AntiVirus Client Installs the Symantec AntiVirus Client silently with no reboot to the target computer. The client only includes the following features:
  • All decomposers but Ghost images
  • All help files
  • Email protection plug-ins for Lotus Notes and Microsoft Exchange/Outlook
  • Scan and Deliver
  • File system real-time protection
  • Live Update
