Symantec Quarantine Service API
===============================

This directory contains files from Symantec that provide a programming
interface to the Quarantine Service.  These files are collected from 
these directories on DISDEBUG CD#2 containing Symantec build 13, dated 6/2/99:

   NAVCORP\CLIENT\Norton_AntiVirus\QuarantineServer\shared\BinIntelAnsiWin32Debug
   NAVCORP\CLIENT\Norton_AntiVirus\QuarantineServer\shared\BinIntelUnicodeWin32Debug
   NAVCORP\CLIENT\Norton_AntiVirus\QuarantineServer\shared\include
   NAVCORP\CLIENT\Norton_AntiVirus\QuarantineServer\shared\LibIntelAnsiWin32Debug
