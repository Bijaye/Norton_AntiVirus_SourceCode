AVIS Web Communication

AVIS Web Communication settings determine how the Quarantine communicates with the gateway.

The Status interval is frequency at which the gateway is polled for status changes about submitted samples.

To specify AVIS Web Communication settings:

  1. In the left pane of the Symantec System Center console, right-click Symantec AntiVirus Quarantine.

  2. Choose Properties and click the AVIS Web Communication tab.
