Using the Toolbar Uparrow button, submit selected files to the Analysis Center.

<P>

For a sample to be eligible for manual submission, it must meet the following criteria.

1. the sample is not already eligible for automatic submission (X-Sample-Priority == 0)

2. the sample has not yet been submitted (X-Date-Submitted == 0)

3. the sample has not yet been analyzed (X-Date-Analysis-Finished == 0)
