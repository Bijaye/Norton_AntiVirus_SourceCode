This directory contains binary builds of snavapi. The builds are not guaranteed
to be using the most recent version of NAVAPI, or built with the most recent
version of snavapi. These are meant as a convinience to uses so they do not have
to build (or request a build) of snavapi. If a user wants a specific build of 
snavapi, or the newest build, they must build snavapi themselves, still.
